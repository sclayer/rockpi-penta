# ROCK Pi Penta SATA

[Penta SATA HAT](<https://wiki.radxa.com/Penta_SATA_HAT>) for ROCK Pi 4B

![penta-hat](https://cos.setf.me/assets/images/penta-sata-hat.png)

Build the package
=================

```bash
dpkg-deb --build --root-owner-group rockpi-penta
```

Install the package
===================

```bash
sudo dpkg -i rockpi-penta.deb
```
